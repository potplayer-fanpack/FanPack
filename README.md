# FanPack-Lite
Polish
========

FanPack64 Lite, czyli autorski pakiet konfiguracyjny oparty na wbudowanych filtrach / dekoderach.  
Wszystko jest już skonfigurowane, instalujesz pakiet, uruchamiasz odtwarzacz i zaczynasz oglądać filmy.
